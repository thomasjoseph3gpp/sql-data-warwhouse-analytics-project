Data Analytics Project
Welcome to my ** Data Analytics Project ** repository . This project demonstrates how we use data available in a data warehouse for advanced analytics in order to  derive actionable insights for a data scientist,business analyst and various other teams. It includes all the best practices and processes that a standard industry follows.

Project Requirements
Objective
Perform various advanced Data Analytical Methods on the data available in the gold layer of the Data Warehouse created using Medallion Architecture.


BI: Analytics & Reporting (Data Analytics)
####Objective

Develop SQL- based analyticas to deliver detailed insights into:

Customer Behaviour

Product Performance

Sales Trends

These insights empower stakeholders with key business metrics, enabling strategic decision-making.
#### Methods Used

		1.Change Over Time Trends
		2.Cumulative Analysis
		3.Runnin count of orders and products
		4.Performance Analysis
		5.Part to whole analysis
		6.Data Segmentation
		7.Custom Reporting
